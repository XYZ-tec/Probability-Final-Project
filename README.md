# CrimenMetrics: Global Organized Crime Analysis 2023

## Project Overview
This project performs a statistical analysis of global organized crime using the 2023 Global Organized Crime Index. We aim to predict criminality based on institutional resilience and illicit market factors.

## Data Baseline (Member 1 Findings)
- **Primary Dataset:** `data/cleaned_crime_data.csv`
- **Key Variables:** Criminality avg, Resilience avg, Human trafficking, Cyber-dependent crimes, State-embedded actors.
- **Current Stats:** 
  - Global Criminality Mean: 5.03
  - Global Resilience Mean: 4.82
  - **95% Confidence Interval for Criminality:** [4.84, 5.22]

## Team Structure
- **Member 1 (Lead):** Data Architect & Statistical Baseline
- **Member 2 & 3:** Probability Distributions & Regression Modeling
- **Member 4 & 5:** Backend API & Web Interface (Streamlit/Flask)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run baseline analysis: `python analysis.py`
